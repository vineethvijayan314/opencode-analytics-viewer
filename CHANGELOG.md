# Changelog

All notable changes are documented here. Releases use semantic versioning.

## 0.1.0 - Unreleased

### Added

- Cross-platform install and launch scripts for macOS, Linux, and Windows.
- Global cost totals in the OpenCode sidebar and footer, refreshing every five seconds.
- Dashboard filters for today and the current Monday-start week.
- Dashboard spend and usage highlight cards.
- Clone-based update scripts.

### Changed

- Dashboard API default is port `7123`.
- OpenCode database lookup supports Windows `LOCALAPPDATA` and Linux/macOS XDG data paths.
